# CamperDays Landing Microservice

## Install Application

install node in version 4.2.X

install dependencies

```
> npm install
```

install global dependencies

```
> cd scripts/ && sh install.sh
```

## Start Application

### Default

```
> npm start
```

### Run in Dev Mode

```
> npm run dev
```

## Develop

```
> npm run watch
```

```
> npm run deploy
```

```
> npm run font
```
